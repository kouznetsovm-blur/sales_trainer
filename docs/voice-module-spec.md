# ТЗ: Модуль голосового ввода
## Приложение: Zori — голосовой тренер для продавцов

---

## 1. Общие принципы

- Всё взаимодействие — стриминговое, без задержек на буферизацию
- Язык: только русский
- Режим: двусторонний голосовой диалог без нажатия кнопок в процессе общения

---

## 2. Голосовой ввод (пользователь → система)

- Захват микрофона начинается сразу после нажатия кнопки "плей"
- Используется VAD (Voice Activity Detection) — система сама определяет, когда пользователь начал и закончил говорить
- Речь стримингом передаётся на сервер для распознавания
- Распознанный текст отображается в транскрипте в реальном времени (слово за словом)

---

## 3. Голосовой вывод (AI → пользователь)

- AI начинает отвечать сразу после окончания реплики пользователя (детектируется VAD)
- Ответ AI синтезируется и воспроизводится стримингом — первые слова звучат до того, как весь текст сгенерирован
- Параметры голоса AI:
  - Пол: мужской
  - Возраст: ~40 лет
  - Тембр: низкий тенор

---

## 4. Обработка перебивания (interruption)

- Если пользователь начинает говорить во время реплики AI:
  - Воспроизведение голоса AI останавливается немедленно
  - Генерация ответа AI прерывается
  - Система переходит в режим прослушивания пользователя

---

## 5. Контент диалога

- AI общается с пользователем на любые темы без заданной роли или сценария
- Диалог не сбрасывается при паузе сессии — при повторном нажатии "плей" контекст разговора сохраняется

---

## 6. Технический стек

- Провайдер: **OpenAI Realtime API** (WebRTC или WebSocket)
- VAD: встроенный в OpenAI Realtime API
- STT: встроенный в OpenAI Realtime API (Whisper, русский язык)
- TTS: встроенный в OpenAI Realtime API
- Модель: **`gpt-realtime-mini`** (GA, актуальная)
  - Контекст: 32k токенов
  - Стоимость: ~$0.016/мин (аудио вход) + ~$0.06/мин (аудио выход)
  - Причина выбора: оптимальное соотношение цены и качества для MVP
  - Возможная миграция: на `gpt-realtime` при необходимости более высокого качества голоса
